## 🌟 Overview

**Keenkeeper** is a modern relationship management tool designed to help you stay connected with the people who matter. In a world of fleeting digital interactions, FriendShelf provides a dedicated space to track your meaningful conversations, monitor the "health" of your friendships, and gain insights into your social habits.

## ✨ Key Features

### 📅 Unified Interaction Timeline
Never lose track of when you last spoke. The **Meaningful Timeline** provides a unified feed of all your interactions—whether they were calls, texts, or video chats—with full filtering capabilities to help you review your history.

### 💖 Relationship Health Tracking
Identify which connections need attention at a glance. Our intelligent dashboard highlights friends who might need a check-in, ensuring that no important relationship falls through the cracks.

### 📊 Friendship Analytics
Visualize your social life with interactive data dashboards. Powered by **Recharts**, FriendShelf categorizes your interactions and shows you exactly how you're spending your social energy.

## 🚀 Technologies Used

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router & React 19)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/) for a premium, responsive UI
- **Icons:** [Lucide React](https://lucide.dev/) for crisp, modern iconography
- **Charts:** [Recharts](https://recharts.org/) for beautiful data visualization
- **Notifications:** [React Toastify](https://fkhadra.github.io/react-toastify/) for elegant user feedback

## 🛠️ Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/friendshelf.git
   cd friendshelf
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open the app:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see your shelf!
